# Practical Order Validation

## 📌 Overview

This project validates customer order inputs such as product name, quantity, and price using Python.

## 🚀 Features

* Input validation (product, quantity, price)
* Error handling using exceptions
* Clean modular structure
* Config-driven keys

## 📂 Project Structure

```
practical_order_validation/
│── config/settings.py
│── src/calculate.py
│── src/main.py
│── requirements.txt
```

## ▶️ Run the Project

```
python -m src.main
```

## 🧪 Sample Output

```
Total for Laptop is 100000
Error: Invalid quantity for Mobile
Error: Invalid price for Headphones
Total for Keyboard is 8000
Error: Missing product in order 5
```
